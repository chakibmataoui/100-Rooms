Engine document

Breif introduction :

The design of the engine should fit the exact requirments of the game so that we can add more performant features to the game.
The modules needed for a game like this are simple :

  • Game Box
  • Player Interactions
  • Game Logic
  • Graphical
  • Sound
  • Text
  • Ressource Manager

Game Box :

The Game box is the actual window where the game is displayed and rendered the player should be able to switch between full and windowed screen in the option menu inside the game and it should contain the game loop and level managment

Player Interactions

This should manage the input of the user so he can interact with the game the UI is a part of the graphical engine so the mouse interaction is coded inside the GameBox i guess for now we'll see later

Game Logic

Since the game logic has its own module for scripting the game so it should contain all what we need to manage the init update draw loop
